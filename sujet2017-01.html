Sujet 1 - 2017

Important : Dans le répertoire Bac 2021, créer un dossier de travail portant votre numéro d'inscription (6 chiffres) et dans lequel vous devez enregistrer, au fur et à mesure, tous les fichiers solutions de ce sujet.

Afin de gérer le transport de marchandises d'une société pour ses clients dans les différentes régions, on se propose de présenter les données nécessaires dans le tableau suivant :

  1. A l'aide du logiciel tableur disponible :
    1. Saisir le tableau ci-dessus, à partir de la cellule A1, dans une feuille à nommer "Marchandises" et enregistrer le classeur sous le nom "Transport_March dans votre dossier de travail.
    2. Appliquer la même mise en forme utilisée dans le tableau ci-dessus.
    3. Remplir la colonne "Coefficient" pour chaque client, sachant que :
      • Coefficient = 0,3 dans le cas où le poids est strictement inférieur à 100Kg
      • Coefficient = 1,2 dans le cas où le poids est strictement supérieur à 300Kg
      • Coefficient = 0,8 dans les autres cas
    4. Remplir la colonne "MontantTrans" pour chaque client selon le type de transport, sachant que :
      MontantTrans = Distance * Coefficient * Frais_km
    5. Copier les données de la plage de cellules A1:C3 dans une nouvelle feuille de calcul à nommer "Type_Transp".
    6. Copier les données de la plage de cellules A5:B11 et E5:G11 d'une manière adjacente dans une nouvelle feuille à nommer "Transport" en commençant à partir de la cellule A1
  2. A l'aide du logiciel de gestion de bases de données disponible :
    1. Créer dans votre dossier de travail, la base de données intitulée "Gestion_Transports" décrite par le schéma relationnel suivant :

      Client(CINClient, NomClient, Region, Distance)

      TypeTransp(CodeType, Libelle, Frais_km)

      Transport(CodeTrans, CINClient, DateTrans, Poids_kg, CodeType)

    2. Importer les données de la table "TypeTransp" à partir de la feuille de calcul "Type_Transp".
    3. Importer les données de la table "Transport à partir de la feuille de calcul "Transport.
    4. Créer la table "Client" et la remplir par les données représentées dans le tableau suivant :
      Client
      CINClient NomClient Region Distance
      01222222 Ste Electro Béjà 100
      02333333 Ali Feki Sfax 270
      03444444 Yemen Triki Sfax 270
      04555555 Ste Soft Gabès 400
    5. Créer les relations entre les tables de cette base de données.
    6. Créer les requêtes suivantes :
      • R1 : Afficher la liste marchandises transportées (NomClient, Région, Poids) dans la journée du "11/04/17" et dont le code de transport est "Tr200"
      • R2 : Afficher la liste des marchandises transportées (NomClient, Poids) dans la journée du "11/04/17" vers la région de "Sfax".
      • Afficher le nombre de clients de la région de Sfax en utilisant la commande Totaux
