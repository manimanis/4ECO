Gare de Sousse

Travail demandé

  1. Télécharger, puis ouvrir la Feuil1 du classeur Série03.xlsx
  2. Déterminer le Prix de Billet (Utiliser les références absolues) :
    • Prix de Billet = 8 ; Si Type = ‘A’
    • Prix de Billet = 13,5 ; Si Type =’AR’
  3. Calculer le Total :

    Total = Prix Billet x Billets Vendus

  4. Calculer la Taxe (Utiliser les références absolues) :
    • Taxe = 1 ; Si type = ‘A’
    • Taxe = 2 ; Si Type =’AR’
  5. Calculer le Montant Taxe :

    Montant Taxe = Taxe x Billets Vendus

  6. Calculer le montant Net des ventes :

    Net = Total – Montant Taxe

  7. Calculer le nombre de billets vendus de chaque type.
  8. Calculer le maximum, le minimum et la moyenne des ventes.
  9. Dessiner un graphique en histogramme pour les ventes de billets.

Ventes d'un magasin

Travail demandé

  1. Ouvrir la Feuil2 du classeur précédent.
  2. Calculer le Stock Final :

    Stock Final = Stock Initial – Ventes

  3. Calculer le nombre d’articles à commander (Commande):
    • Commande = 15 ; Si Stock Final ≤ Seuil et Type = ‘A’
    • Commande = 30 ; Si Stock Final ≤ Seuil et Type = ‘B’
    • Commande = 0 ; Si Stock Final > Seuil
  4. Calculer le Nouveau Stock :

    Nouveau Stock = Commande + Stock Final

  5. Déterminer les articles en surstock :
    • Surstock = ‘Oui’ ; Si Nouveau Stock ≥ 2 * Seuil
    • Surstock = ‘Non’ ; Si Nouveau Stock < 2 * Seuil
  6. Déterminer les articles en promotion :
    • Promo = -20% ; Si Type = ‘B’
    • Promo = 0% ; Si Type =’A’
  7. Calculer les totaux.
  8. Calculer l’état de ventes :
    • Etat de vente = ‘Acceptable’ ; Si Totaux Ventes ≥ 75% × Stock Initial
    • Etat de vente = ‘Critique’ ; Si Totaux ventes < 75% × Stock Initial
  9. Représenter graphiquement, en courbe, les ventes et le Stock Final.

Notes des élèves

Travail demandé

  1. Ouvrir la feuille Feuil3 du classeur précédent.
  2. Calculer la moyenne de chaque élève (utiliser les références absolues)
  3. Calculer la décision :
    • Décision = ‘Admis’ ; Si Moyenne ≥ 10 et Sexe = ‘G’
    • Décision = ‘Admise’ ; Si Moyenne ≥ 10 et Sexe = ‘F’
    • Décision = ‘Refusé’ ; Si Moyenne < 10 et Sexe = ‘G’
    • Décision = ‘Refusée’ ; Si Moyenne < 10 et Sexe = ‘F’
  4. Calculer la moyenne de la classe
  5. Calculer le nombre de filles et le nombre de garçons.
  6. Mettre en forme le tableau
  7. Représenter graphiquement les notes des élèves (graphique courbe).

Location de voitures

Travail demandé

  1. Ouvrir la feuille Feuil4 du classeur précédent.
  2. Calculer la durée de location :

    Durée Location = Date Sortie – Date Entrée

  3. Calculer les Frais HT :
    • Frais HT = 60 × Durée Location ; Si Durée Location ≤ 5 jours
    • Frais HT = 50 × Durée Location ; Si Durée Location ≤ 10 jours
    • Frais HT = 45 × Durée Location ; Si Durée Location > 10 jours
  4. Calculer la TVA :

    TVA = Frais HT × TVA%

  5. Calculer les Frais TTC :

    Frais TTC = Frais HT + TVA

  6. Calculer le Total.
  7. Mettre en forme le tableau, mettre les montants avec trois décimales.
  8. Représenter graphiquement la durée de location en fonction du CIN.
